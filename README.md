# Vue 项目名称

vue3-audio

## 介绍

这是一个基于 vue3 写的 audio 插件
以下是参数：
ttsAudioUrl:audio 的 url
isPauseTtsAudio:是否暂停播放试听 默认 true
forwardSecond:快进速度 默认 10
backSecond:快退速度 默认 3
color1:背景渐变颜色从左往右 默认 9cafe8
color2:背景渐变颜色 默认 9cebb8
